Алгоритм состоит в следующем:
Будем называть слово хорошим, если оно содержит префикс искомой буквы какой-либо длины
Будем хранить на стеке парсинга тройку:
1. Сет из пар - длина хорошего префикса  из языка и длина самого слова для всех хороших слов из языка
2. Минимальная длина слова из языка
3. Сам ответ на задачу
Соответственно, эти величины пересчитываются используя друг друга следующим образом:
Конкатенация:
Просто пробегаемся по сету и прибавляем к первому слову либо все слова из второго сета (если оно состоит только из помеченной буквы), либо только минимальное слово из второго, параллельно обновляя ответ
Объединение:
Просто объединяем сеты, берем минимум из ответов как ответ.
Звезда:
Раскрываем по формуле через объединение и степень языка.
