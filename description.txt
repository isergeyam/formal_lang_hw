Алгоритм состоит в следующем:
Будем называть слово хорошим, если оно содержит префикс искомой буквы какой-либо длины
Будем называть хорошим префиксом хорошего слова максимальный префикс, состоящий только из помеченной буквы.
Будем хранить на стеке парсинга тройку:
1. Сет из пар - длина хорошего префикса и длина самого слова для всех хороших слов из языка
2. Минимальная длина слова из языка
3. Сам ответ на задачу
Соответственно, эти величины пересчитываются используя друг друга следующим образом:
Конкатенация:
Построение сета:
Пробегаемся по сету первого языка. Если текущее слово - хороший префикс, то конкатенируем его со всеми хорошими словами из второго языка. Иначе, конкатенируем только с минимальным словом.
Для каждого посчитанного нового хорошего слова пытаемся обновить ответ, который сначала - бесконечность.
Минимальное слово - сумма минимальных.
Объединение:
Просто объединяем сеты, берем минимум из ответов как ответ. Минимальное - также минимум.
Звезда:
Раскрываем по формуле через объединение и степень языка.
